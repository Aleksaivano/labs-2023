Все константы в дефайнах пишем через Upper Case нпример SOME_CONST
Названия переменных в camalCase, название функций и типов структур в CamalCase
Между операндом и бинарным оператором обязательно пробел, между операндом и унарным оператором пробел не ставим.
Примеры оформления условных конструкций:
if (<условие>) {
   ...
} else if (<другое условие>) {
  ...
} else {

}

Пример оформления циклов:
for (int i = 0; i < 10; i++) {
    ...
}


while (i < 10) {
      ...
}

Пример оформления функций:

int increment(int x) {
    return x + 1;
}

Пример вызова функций:
increment(5);

Табуляцию и вложенность соблюдаем и названия переменных, констант и функций по возможности выбираем понятные, a, b, c используем там, где это так или иначе обосновано.
